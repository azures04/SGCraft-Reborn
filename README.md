# 🌌 SGCraft Reborn

![Assets License](https://img.shields.io/badge/Assets-MIT-brightgreen.svg)

**SGCraft Reborn** is a complete, from-scratch rewrite and modernization of the legendary Minecraft mod *SGCraft* (originally created by gcewing). Build massive Stargates, dial addresses across dimensions, and explore the universe!

---

## ✨ Features

- **Fully Functional Stargates:** Construct 5x5 multiblock Stargates to travel instantly between locations and dimensions.
- **Dial Home Device (DHD):** Quickly dial your saved addresses using the iconic DHD interface.
- **The "Kawoosh" Effect:** Experience the beautifully recreated, massive liquid-piston event horizon animation when a gate opens.
- **Waterlogging Support:** Perfect for aquatic gate rooms! All Stargate components now support waterlogging, allowing you to build and operate your gate entirely underwater.
- **Base Defense:** Craft and install an Iris to protect your gate room from unwanted visitors.
- **Naquadah Tech Tree:** Mine Naquadah ore and refine it into ingots and core crystals to build your gate network.
- **Power Integration:** Compatible with modern energy systems (RF / Forge Energy) via the Power Unit block.
- **Computer Automation:** Full integration with **ComputerCraft** and **OpenComputers**. Write scripts to automatically dial addresses, manage the iris, and read gate states!

## 📥 Installation

1. Ensure you have **Minecraft Forge** installed for the correct version.
2. Download the latest `SGCraft-Reborn` `.jar` file from the Releases page.
3. Drop the `.jar` into your Minecraft `mods` folder.
4. Launch the game and start building your gate room!

## 🛠️ Building from Source

If you want to compile the mod yourself or contribute to the development:

1. Clone the repository:
   ```bash
   git clone [https://github.com/azures04/sgcraft-reborn.git](https://github.com/azures04/sgcraft-reborn.git)
   cd sgcraft-reborn
   ```
2. Set up the Forge workspace and build the project using Gradle:
    ```bash
    # On Windows
    gradlew build
    
    # On Linux / macOS
    ./gradlew build
    ```
3. The compiled `.jar` file will be located in `build/libs/`.
